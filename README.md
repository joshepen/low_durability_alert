# Low Durability Alert: A DST Mod
I keep losing magis to forgetting to watch the durability, so I made this client mod to play a sound when any equipped item falls below some durability amount. The percents and sounds are configurable in the mod config options.

## Installation
Clone the repo and place it in DST's `mods/` folder.

## Usage
Enable it in the client mod menu and configure to your liking
